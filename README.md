### Installation üíª

##### Step 1 - Clone the repository to your local machine using Git:

```bash
git clone https://github.com/Adi3042/Churn-Prediction-ANN.git
cd Churn-Prediction-ANN
```

##### Step 2 - Create a conda environment after opening the repository

```bash
conda create -n venv python==3.11 -y
conda activate venv/
```

##### Step 3 - Install the requirements

Open your terminal and execute the following command:

```bash
pip install -r requirements.txt
```

##### Step 4 - Run the application server

Open your terminal and execute the following command:

```bash
streamlit run app.py
```

##### Step 5 -

1. Visit the web app. :- http://127.0.0.1:8501/
2. Enter the attributes in the input form.
3. Receive the predicted Churn.
